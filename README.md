# Franco Sebastian Arraño Silva - Portfolio

Personal portfolio built with HTML, CSS, and vanilla JavaScript.

## 🚀 Features

- Modern, responsive design
- Dark/Light mode toggle
- Animated sections on scroll
- Parallax hero effect
- Projects showcase with tech tags
- Skills visualization
- Floating contact button

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styles with CSS variables
- **JavaScript** - Vanilla JS for interactions
- **Animate.css** - Animations library
- **Font Awesome** - Icons
- **Plus Jakarta Sans** - Typography

## 📱 Responsive

Works on all devices: mobile, tablet, and desktop.

## 🚦 Deployment

Automatically deployed to GitHub Pages via push to master branch.
